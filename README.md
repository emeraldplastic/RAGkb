# RAGkb

RAGkb is a private document Q&A app with a FastAPI backend, React frontend, per-user document isolation, and retrieval-grounded streaming answers.

## Highlights

- Password auth with JWT issuer and audience validation
- Per-user SQLite metadata, upload folders, and Chroma collections
- PDF, TXT, and Markdown ingestion with queued processing status
- Streaming chat responses with visible source references
- Strict API no-store responses, security headers, trusted hosts, and rate limits
- Account and document deletion endpoints
- Vercel deployment configuration

## Local Development

1. Install backend dependencies:

```bash
pip install -r requirements.txt
```

2. Install frontend dependencies:

```bash
cd frontend
npm install
```

3. Create `.env` from `.env.example` and set an AI provider key.

4. Run the backend:

```bash
uvicorn main:app --reload
```

5. Run the frontend in another terminal:

```bash
cd frontend
npm start
```

## Test

pytest -q

## Deploy On Vercel

This repo includes `vercel.json` and `api/index.py`. Build the React app before deploying so `frontend/build` is available to the Python function bundle.

Required production environment variables:

- `APP_ENV=production`
- `SECRET_KEY` generated by Render or set manually
- `AI_PROVIDER` as `openai` or `google`
- `OPENAI_API_KEY` for OpenAI or OpenAI-compatible providers
- `GOOGLE_API_KEY` for Gemini
- `DATABASE_PATH`, `CHROMA_PATH`, and `UPLOAD_DIR`

Current Vercel mode uses ephemeral function storage, so it is suitable for demos and short-lived sessions. For market-ready persistence on Vercel, move metadata, files, and vectors to managed services such as Neon Postgres, Vercel Blob, and a hosted vector database.

Deploy:

```bash
cd frontend
npm run build
cd ..
vercel --prod
```

## Provider Notes

OpenAI defaults:

```bash
AI_PROVIDER=openai
LLM_PROVIDER=openai
EMBED_PROVIDER=openai
EMBED_MODEL=text-embedding-3-small
LLM_MODEL=gpt-4o-mini
```

Gemini defaults:

```bash
AI_PROVIDER=google
LLM_PROVIDER=google
EMBED_PROVIDER=google
GOOGLE_API_KEY=<your-gemini-api-key>
LLM_MODEL=gemini-1.5-flash
EMBED_MODEL=models/text-embedding-004
```
